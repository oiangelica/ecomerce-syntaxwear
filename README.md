# SyntaxWear

SyntaxWear é um modelo de landing page de e-commerce moderno e responsivo para uma loja de sapatos e tênis. Este projeto foi construído com HTML e CSS, sem o uso de JavaScript, tornando-o leve e de carregamento rápido.

![GIT de apresentação do projeto](./src/images/readme/preview-readme.gif)

## Características

- **Design Responsivo:** O layout se adapta a diferentes tamanhos de tela, proporcionando uma ótima experiência de usuário em desktops e dispositivos móveis.
- **UI Moderna:** Design limpo e moderno seguindo as últimas tendências.
- **Apenas CSS:** O site inteiro é estilizado com CSS, sem dependências de JavaScript.
- **CSS baseado em componentes:** O CSS é organizado em componentes, facilitando a manutenção e escalabilidade.
- **Seção Hero:** Uma seção de hero proeminente para mostrar os principais produtos ou promoções.
- **Categorias de Produtos:** Uma seção para exibir as diferentes categorias de produtos.
- **Grid de Produtos:** Uma grade para mostrar os produtos em destaque.
- **Rodapé:** Um rodapé abrangente com um formulário de inscrição para newsletter, links de mídia social e navegação.

## Como Usar

1. Clone este repositório: `git clone https://github.com/oiangelica/ecomerce-syntaxwear.git`
2. Abra o arquivo `index.html` no seu navegador.

## Customização

Você pode personalizar facilmente o site editando os arquivos CSS no diretório `src/css`. O arquivo `src/css/components/variables.css` contém todas as variáveis de cor e fonte, facilitando a alteração do esquema de cores do site.
